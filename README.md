# Basic-Calculator-v0.5


Basic Calculator v0.5 is a beginner-friendly Python console application designed to perform essential mathematical operations with clean input-validation and clear user guidance.

Features

Addition

Subtraction

Multiplication

Division (with zero-division protection)

Square Root (single-number operation)

Power (base + exponent)

Highlights

Fully English interface

Continuous input validation (program never crashes on invalid input)

Simple loop system for repeated operations

Clear messages, structured logic

Easy to understand for new learners
